# Organizacao de papers

Estrutura atual:

- `pdfs/`: corpus preferencial dos papers.
- `ui/`: interface web local para fazer perguntas ao RAG.
- `docs/`: documentacao de uso.
- `rag_index/`: artefatos do indice persistido.
- `rag_papers.py`: ponto de entrada do backend e da interface.

Nota: alguns PDFs antigos ainda permanecem na raiz desta pasta porque o ambiente bloqueou a movimentacao pelo terminal. O backend continua lendo esses arquivos automaticamente.
